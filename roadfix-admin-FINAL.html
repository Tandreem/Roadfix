🔥 Firebase setup checklist

1
Authentication enabled — Firebase console → Build → Authentication → Email/Password → Enable
2
Realtime Database enabled — Build → Realtime Database → Test mode → Enable
3
All 3 files uploaded to github.com/lakhyajit/roadfix-demo with correct Firebase config
4
Admin login — use same Google account email as your Firebase project owner, or create a user in Firebase Auth with any email/password
Firebase data structure
/customers/{uid} → {name, phone, city, email, role, createdAt}
/mechanics/{uid} → {name, phone, skills, status, gpsOn, bodycam, verified, lat, lng}
/jobs/{jobId} → {customerId, customerName, issue, status, mechanic, partType, estimateLow, estimateHigh, finalPrice}
/alerts/{alertId} → {type, name, phone, location, lat, lng, time, status, app}

job status flow:
pending → searching_mechanic → mechanic_assigned → enroute → fixing → done

alert types: customer_sos · mechanic_sos
